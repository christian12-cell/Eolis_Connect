# Manuel Utilisateur — Clients
## Eolis Connect — Eolis Cameroun

---

## 1. Accéder à l'application

### Via navigateur web
Ouvrez votre navigateur et allez sur : **https://eolisconnect.online**

### Installer l'app sur votre téléphone (recommandé)

**Sur Android (Chrome) :**
1. Ouvrez https://eolisconnect.online dans Chrome
2. Appuyez sur le menu `⋮` en haut à droite
3. Sélectionnez **"Ajouter à l'écran d'accueil"**
4. Appuyez sur **Ajouter**
5. L'icône Eolis Connect apparaît sur votre écran d'accueil

**Sur iPhone (Safari uniquement) :**
1. Ouvrez https://eolisconnect.online dans **Safari**
2. Appuyez sur le bouton partage ↑ en bas de l'écran
3. Faites défiler et appuyez sur **"Sur l'écran d'accueil"**
4. Appuyez sur **Ajouter**

**Sur ordinateur (Chrome/Edge) :**
1. Ouvrez https://eolisconnect.online
2. Cliquez sur l'icône d'installation ⊕ dans la barre d'adresse
3. Cliquez **Installer**

---

## 2. Créer un compte

1. Sur la page d'accueil, cliquez **"Créer un compte"** ou **"S'inscrire"**
2. Remplissez les champs :
   - **Prénom et Nom**
   - **Email** (adresse valide — vous recevrez des notifications ici)
   - **Téléphone** (optionnel — pour recevoir des SMS)
   - **Mot de passe** (minimum 8 caractères)
3. Cliquez **"Créer mon compte"**
4. Vous êtes connecté automatiquement

---

## 3. Se connecter

1. Entrez votre **nom d'utilisateur** ou **email**
2. Entrez votre **mot de passe**
3. Cliquez **"Se connecter"**

**Mot de passe oublié ?**
1. Cliquez **"Mot de passe oublié"**
2. Entrez votre email
3. Suivez les instructions reçues par email

---

## 4. Soumettre une demande (ticket)

Une demande = un dossier ouvert pour un problème ou une question.

1. Depuis votre tableau de bord, cliquez **"Nouvelle demande"**
2. Remplissez le formulaire :
   - **Catégorie** : choisissez le type de votre problème
   - **Sous-catégorie** : précisez davantage
   - **Urgence** : 
     - 🔴 **Élevée** — problème bloquant urgent
     - 🟡 **Moyenne** — problème important non urgent
     - 🟢 **Faible** — question ou demande simple
   - **Description** : décrivez clairement votre problème
   - **Pièces jointes** : ajoutez des photos ou documents si nécessaire
3. Cliquez **"Soumettre"**
4. Votre dossier est créé avec un **numéro de référence** (ex: ELS-2026-001)

---

## 5. Suivre vos demandes

1. Cliquez sur **"Mes demandes"** dans le menu
2. Vous voyez la liste de tous vos dossiers avec leur statut :
   - **En attente** : votre demande est enregistrée, un agent va la prendre en charge
   - **En cours** : un agent traite votre demande
   - **Clôturé** : votre demande est résolue

3. Cliquez sur un dossier pour voir les détails et les messages échangés

---

## 6. Communiquer avec un agent

Une fois votre dossier ouvert, vous pouvez échanger des messages avec l'agent :

1. Ouvrez votre dossier
2. Faites défiler vers le bas jusqu'à la zone de message
3. Tapez votre message
4. Cliquez **Envoyer**

**Types de messages que vous pouvez recevoir :**
- **Message de l'agent** : réponse ou question de l'agent
- **Demande de documents** : l'agent vous demande des fichiers spécifiques
- **Réponse finale** : l'agent a résolu votre problème — lisez bien ce message

**Pour envoyer un fichier :**
1. Cliquez sur l'icône trombone 📎
2. Sélectionnez votre fichier
3. Cliquez **Envoyer**

---

## 7. Les notifications

Vous recevrez des notifications quand :
- Un agent répond à votre demande
- L'agent vous demande des documents
- Votre demande est résolue

**Dans l'app :** la cloche 🔔 en haut affiche le nombre de messages non lus.

**Par email :** vous recevrez un email à chaque mise à jour importante.

---

## 8. Évaluer le service

Après la clôture d'un dossier, vous pouvez noter la qualité du service :

1. Ouvrez le dossier clôturé
2. Cliquez sur les étoiles ⭐ (1 à 5)
3. Ajoutez un commentaire si vous le souhaitez
4. Cliquez **"Envoyer mon avis"**

Vos évaluations aident à améliorer la qualité du service.

---

## 9. Mode hors ligne

L'application fonctionne partiellement sans connexion internet :

✅ **Ce qui fonctionne hors ligne :**
- Consulter vos demandes déjà chargées
- Lire les messages déjà téléchargés

❌ **Ce qui nécessite une connexion :**
- Créer une nouvelle demande
- Envoyer un message
- Recevoir de nouvelles notifications

Quand vous retrouvez la connexion, l'app se synchronise automatiquement.

---

## 10. Changer la langue

1. Sur la page de connexion, cliquez sur **FR** ou **EN** en haut
2. La langue change immédiatement

---

## 11. Modifier votre profil

1. Cliquez sur votre nom ou l'icône profil
2. Allez dans **"Paramètres"**
3. Modifiez vos informations (prénom, nom, téléphone, langue)
4. Cliquez **"Sauvegarder"**

---

## 12. Questions fréquentes

**Mon dossier est-il traité dans l'ordre ?**
Les dossiers urgents (🔴 Élevée) sont traités en priorité.

**Combien de temps pour une réponse ?**
- 🔴 Élevée : objectif < 3 heures
- 🟡 Moyenne : objectif < 5 heures
- 🟢 Faible : objectif < 10 heures

**Je ne reçois pas les emails de notification ?**
Vérifiez votre dossier spam/courrier indésirable.

**Je n'arrive pas à me connecter ?**
Utilisez la fonction "Mot de passe oublié" ou contactez le support.
